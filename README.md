### REST API use Express.js and MariaDB
### Usage
### GET USERS (GET)
```js
    http://localhost:4000/api/user
```
###
### GET USER (GET)

```js
    http://localhost:4000/api/user/id
```
###
### CREATE USER (POST)
```js
    http://localhost:4000/api/user
```
###
### UPDATE USER (PUT)

```js
    http://localhost:4000/api/user/id
```
###
### DELETE USER (DELETE)

```js
    http://localhost:4000/api/user/id
```
